04 How it works

The hackathon grades the harness, not the agent. The agent stays dumb — it only ever sees system + messages + feedback. The orchestrator drives the loop; the four pillars do the constraining, around a swappable worker.

ORCHESTRATOR (Run Engine) — thin sequencer, no domain logic state machine · retry loop · journal · replay · escalation ┌──────────┬────────────┬─────────────┬───────────────┬─────────────┐ │ MATERIAL │ GUARDRAILS │ CHECKPOINTS │ OBSERVABILITY │ PERSONA │ │ in/out │ (persona) │ (gates) │ & ALARMS │ STORE │ └──────────┴─────┬──────┴─────────────┴──────┬────────┴─────────────┘ │ system + messages + feedback (only) ▼ ┌─────────────────────────────┐ ── critical alarm ──▶ HUMAN-IN-THE-LOOP │ AGENT (worker, SWAPPABLE) │ │ research → build → refine │ Mock · Vercel-SDK · native+web_search └─────────────────────────────┘

The control loop — where behavior changes

persona     = PersonaStore.load(id)
guardrails  = compile(persona)        // prompt + validators
material    = Source.load(concept, persona)

research = agent.research({system, concept})
if research-sufficiency FAILS → feedback → research again

webpage  = agent.build({system, research})
for gate in [voice, design, quality]:
    r = gate.evaluate(ctx); journal.append(r)   // replay point
    if r.FAILED && autoCorrectable:
        webpage = agent.build({..., feedback})   // ◀ behavior changes
    elif CRITICAL: escalate()                    // stop & ask

Sink.publish(webpage)   // only voice-faithful, researched, quality pages ship

The four pillars, mapped

PillarGuarantees
Materialclean in (concept + persona) / out (self-contained webpage)
Guardrailsoutput sounds & looks like the persona
Checkpointsexplicit pass/fail + replay; a fail is a real outcome
Observabilitystructured alarms: type · severity · context · action

Two planes

Control plane = Next.js UI on Vercel (onboarding · run stream · escalation · preview). Harness plane = Node/TS backend + SQLite, reached via an ngrok tunnel. The agent is just an outbound API call — local now, cloud-deployable later.
